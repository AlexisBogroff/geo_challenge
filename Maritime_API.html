Preligens Maritime API documentation

Authentication

Get API key

Get a key from your login and password for later use when calling the services.

Request JSON schema: 
- username: string
- password: string

POST https://preligens.com/marineapi/vfake/auth/getAPIkey
Example Payload:

{
“username”: “xxxx”,
“password”: “yyyy”
}

Response codes
Code 200 : OK
Code 400: Incorrect request
Code 401: Unkown credentials

Sample response

{
“APIkey:” PP-RR-EE-LL-II-GG-EE-NN-SS”,
“validUntil”: “2021-10-01”
}

Ship GPS positions

Get the GPS positions of detected ships in port.

Query parameters

  • port_ID : string.
  • Start time: string < datetime >
  • End time: string < datetime >

Header parameters

  • API Key: string

GET https://preligens.com/marineapi/vfake/auth/getShipPositions

Response codes
Code 200 : OK
Code 400: Incorrect request
Code 401: Unkown credentials
